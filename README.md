# Uni-timetable
주제 선정 이유
  - uni-timetalbe(UTT)는 학교를 다니는 학생들 그중 특히 대학생들을 위한 시간표 앱으로
    주로 강의 시간표와 매 수업에 관한 To-do list기능을 제공한다.
    
개발 환경
  - 언어: Kotlin
  - 개발 툴: Android studio
  - 아키텍쳐 패턴: MVC
  - 라이브러리: Jetpack(Room, Viewbinding)

주요기능
  - Gesture를 통한 시간표 생성 및 수정 기능
  - To-do list 형식의 과제 추가 및 삭제 기능
  - 강의, 출석, 과제 배너 알람 설정 기능
  - 다양한 색상 설정 기능
